To connect to the application, the following steps must be followed in the order shown:

FRONTEND
First, we must open the folder that says recovery-frontend, then we enter these commands in the Visual Studio Code console.
1. npm install -g @angular/cli
2. ng config -g cli.warnings.versionMismatch false
3. npm install
4. ng serve 


BACKEND
Next, open the folder that says recovery-backend, then enter these commands in the Visual Studio Code console.
1. npm i
2. nodemon start


MONGODB COMPASS
Finally, we enter the following URI to visualize the stored data: 
URI: mongodb+srv://grupo2:grupo2@cluster0.m2psu.mongodb.net/recoverydb?retryWrites=true&w=majority
